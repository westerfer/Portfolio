Design Mantra

Keep. It. Simple. Stupid.
K.I.S.S. — One of most important rules that I like to follow when developing a user interface or creating that perfect experience. However, there's a certain point in every design where this rule, like many others, is meant to be broken.
 

User Research

Without the guidance of how an individual uses a program, you're essentially navigating without a compass.

 

Visual Design

I like to approach a design with a simple & minimalistic thought. I would much rather prefer the user to focus on the business side of the project rather than the creativeness.

 

Reuse Code

Simplifying and reusing the amount of code in a project can significantly decrease the project timeline. In the long run, reusing code will organically make the project simpler in the end.
